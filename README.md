# ToDo List на React

## Описание: 



## Функционал:

* Авторизация по одноразовому паролю
* Смена аватара и никнейма профиля
* Поиск по пользователям
* Возможность отправить фото,голосовые и текстовые сообщения

## Используемые технологии:

* ReactJS
* HTML/CSS
* TypeScript
* WebPack
* MaterialUI
